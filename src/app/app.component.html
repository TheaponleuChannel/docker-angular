rocket_launch Quick Start Instructions

Choose your preferred path below. Run natively using npm for quick iterations, or isolate completely with Docker.

1 Run Natively with NPM

# Install dependencies
npm install
# Run the development server (auto-opens http://localhost:4200)
npm start

2 Run with Docker Compose

# Build and spin up the container in one command
docker compose up --build -d
# Application will be accessible at:
http://localhost:4200
